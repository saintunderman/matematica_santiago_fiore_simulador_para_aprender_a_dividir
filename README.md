# Simulador para Aprender a Dividir

Aplicación web interactiva diseñada para enseñar y practicar la **división con números naturales**, combinando explicación teórica, resolución paso a paso, ejercicios prácticos, historial de operaciones y modo contrarreloj.

Recurso educativo ideal para estudiantes de nivel primario y primeros años de secundaria.
